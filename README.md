# ticketing

> Customer Support Ticketing SPA System with Vue.js

## Build Setup

``` bash
cd /path/to/ticketing

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run jasmine spec tests
jasmine

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
